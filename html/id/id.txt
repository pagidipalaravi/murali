What is id attribute?
	The HTML id attribute is used to specify a unique id for an HTML element.
	we cannot have more than one element with the same id in an HTML document.
	id attribute used to symbol as "#"(hash).
	NOTE:
		The id name must contain at least one character, cannot start with a number.
	Difference between class attribute and id attribute?
		class name can be used by multiple HTML elements and id name must only be used by one HTML element.
